# 🌍 全球电话号码生成器 (Global Phone Number Generator)

这是一个简单而有趣的网页应用，旨在帮助开发者测试国际电话号码格式，或者仅仅是为了好玩！这是我的第一个学习项目。

## ✨ 功能特点

*   **多国支持**：支持生成中国 (CN)、美国 (US)、英国 (UK)、日本 (JP)、德国 (DE) 等多国电话号码格式。
*   **一键生成**：点击按钮即可随机生成符合当地规范的号码。
*   **语音朗读**：使用 Web Speech API，用当地语言朗读生成的电话号码（体验多语言环境！）。
*   **精美 UI**：现代化的界面设计，响应式布局。

## 🚀 如何使用

1.  直接打开 `index.html` 文件。
2.  在下拉菜单中选择一个国家。
3.  点击 "生成号码" 按钮。
4.  点击 🔊 图标听取号码读音。

## 🛠️ 技术栈

*   HTML5
*   CSS3 (Modern Design)
*   JavaScript (ES6+)
